# Progate-java4
### クラスの定義
class クラス名 {
}
### インスタンスの生成
クラスからインスタンスを生成するには「new クラス名()」とする。
class Main {
  public static void main(String[]args) {
    new Person();
    (personクラスのインスタンスを生成)
  }
}
### インスタンスに変数を代入
変数にインスタンスを代入するには「クラス型 変数名 = new クラス名()」とする。
class Main {
  public static void main(String[]args) {
    Person person = new Person();
    (クラス型 変数名)
  }
}
